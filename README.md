# Pokédex App 🎮

A responsive Pokédex web application built with vanilla JavaScript 
and the PokéAPI. Browse, explore, and catch your favourite Pokémon!

## 🌐 Live Demo
https://dhruvi-pokemon.netlify.app/

## ✨ Features
- Displays first 20 Pokémon on load with thumbnails and names
- **Load More** button to fetch the next 20 Pokémon
- Click any Pokémon to view detailed info (types, abilities, full image)
- **Catch** Pokémon and save them to a caught list
- Caught Pokémon are visually distinguished from uncaught ones
- **Release** Pokémon from your caught list
- Caught list persists using `localStorage` — survives page refresh
- Fully responsive UI built with CSS Grid/Flexbox

## 🛠️ Tech Stack
- HTML5 (semantic markup)
- CSS3 (Flexbox / CSS Grid)
- Vanilla JavaScript (ES6+)
- Fetch API for async requests
- localStorage for persistent data
- [PokéAPI](https://pokeapi.co) — free, no API key needed
